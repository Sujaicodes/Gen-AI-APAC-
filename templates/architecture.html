🔧 System Components

  • Primary Agent: Orchestrates sub-agents and manages complex workflows
  • Sub-Agents: Specialized agents for Search, Task, Calendar, and Notes
  • MCP Tools: Model Context Protocol integrated tools for data operations
  • Database: SQLite with SQLAlchemy for persistent structured data
  • API Layer: FastAPI with REST endpoints and WebSocket support
  • UI Layer: Responsive web interface with real-time updates

⚙️ Technical Stack

  • Backend: Python FastAPI with async/await
  • Database: SQLite with SQLAlchemy ORM
  • Real-time: WebSocket for live agent status
  • Frontend: HTML5, CSS3, JavaScript ES6+
  • Visualization: Mermaid.js for system diagrams
  • Deployment: Uvicorn ASGI server
