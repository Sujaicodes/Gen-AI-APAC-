System Components

  • Primary Agent: Coordinates sub-agents and manages workflows
  • Sub-Agents: Specialized agents for specific tasks (Search, Task, Calendar, Notes)
  • Tools: MCP-integrated tools for calendar, task management, notes, and search
  • Database: SQLite database for persistent storage of tasks, events, and notes
  • API: FastAPI-based REST API and WebSocket for real-time communication
  • UI: Web interface with Chat, Database, and Architecture views

Workflows

  • Plan My Week: Search + Task + Calendar + Notes for weekly brief
  • Proposal Deadline: Task + Calendar + Notes for deadline management
  • Schedule Team Sync: Calendar + Task + Notes for meeting coordination
  • What's On My Plate?: Search + Task + Calendar for daily overview
